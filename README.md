# Libro
 A custom tiddlywiki edition.Libro contains the plugins, theme and custom css/macro that I like to use in all of my wikis. 
  
Macros

* Copy code
* Detail macro

Styles

* Counter for list
* Better fold banner
* Scroll indicator
* Discrete Tags

Buttons

* Save to Github button
* Theme switcher

QOL

* Alternative link
* Description and source templates
* Custom shortcuts
* RSS, images, wiki export
